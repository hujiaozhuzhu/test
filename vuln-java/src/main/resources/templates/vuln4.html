漏洞原理

Shiro的RememberMe功能使用AES加密序列化数据，如果密钥泄露或使用默认密钥，攻击者可以构造恶意的序列化对象

默认密钥

Shiro 1.2.4及以下版本使用硬编码的AES密钥：kPH+bIxk5D2deZiIxcaaaA==

利用工具

推荐工具：shiro_attack、ysoserial生成payload，然后加密生成rememberMe cookie

java -cp shiro_tool.jar com.shiro.tool.Actuator -k "kPH+bIxk5D2deZiIxcaaaA=="
